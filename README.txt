Дудников А.А. 2 курс 1 группа
Задание 15

1. Ввести входные параметры в файл `initial_data.txt`
Сначала вводится вместимость коробок, а затем через пробел вес каждого предмета
Например: 10 3 6 9 1
2. Запустить `Dudnikov_task_15.exe`
3. Посмотреть результат в файле `Решение1.txt` для решения с использованием цикла
или в файле `Решение2.txt` для решения с использованием рекурсии
Первое число - это ответ на задачу для введенных параметров
Второе число - время работы алгоритма, округленное до тысячных
Если введенный вес хотя бы одного предмета будет некорректен, то программа выдаст ошибку в консоли
